Листы и их назначение

📋 АРТИКУЛЫ

Каталог всех карточек товаров из Content API. Обновляй перед остальными загрузками.

  • nmIDУникальный ID карточки WB
  • vendorCodeАртикул, заданный продавцом
  • subjectNameПредмет (подкатегория WB)

📊 ОСТАТКИ_WB

Остатки товаров на складах WB (FBW). Обновляется раз в день, данные на начало суток.

  • quantityТекущий остаток (шт.)
  • inWayToClientВ доставке к покупателю
  • quantityFullПолный остаток (склад + в пути)

🛒 ЗАКАЗЫ

Все заказы с момента указанной даты. Включает отменённые (isCancel=true).

  • priceWithDiscЦена, которую заплатил покупатель
  • isCanceltrue = заказ отменён
  • sridУникальный ID строки заказа

💰 ПРОДАЖИ

Подтверждённые продажи. saleID начинается с "S" — продажа, "R" — возврат.

  • forPayСумма к перечислению продавцу
  • paymentSaleAmountОплата покупателя (с СПП)
  • saleIDID продажи (S.../R...)

🧾 ФИНАНСЫ

Детализация отчёта реализации. Данные с 29.01.2024. Пагинация по rrd_id.

  • rrd_idУникальный ID строки (для пагинации)
  • doc_type_nameПродажа / Возврат / Штраф / Логистика
  • ppvz_for_payК выплате по строке
  • penaltyШтраф (отрицательное = удержание WB)

💳 БАЛАНС

Текущий баланс кабинета. Требует отдельный токен категории "Финансы".

  • currentОбщий баланс на счёте
  • for_withdrawДоступно к выводу прямо сейчас

📦 ПОСТАВКИ

Список поставок FBW за период. Статусы: 1=Черновик, 2=Подтверждена, 3=Принята, 4=Завершена, 5=Отменена.

📉 РАСХОДЫ

Агрегация из ФИНАНСЫ. wb_expense_estimate = Валовая сумма − К выплате.

📘 ДДР

Доходы-Расходы по месяцам. Строится из ЗАКАЗЫ + ПРОДАЖИ + ФИНАНСЫ + РАСХОДЫ.

  • ddr_net_cashflow_estimateК выплате WB − Штрафы (оценка)
  • drr_percentДоля расходов / оборот × 100

📝 ЛОГИ

Журнал запусков: время, длительность, статус (OK/ERROR/PARTIAL), кол-во загруженных строк.

Токены WB API

Категории токенов

Разные методы требуют разные категории токена:

  • КонтентАртикулы (/content/v2/...)
  • СтатистикаЗаказы, продажи, финансы
  • Поставки FBWПоставки (/api/v1/supplies/...)
  • ФинансыБаланс (/api/v1/account/balance)
